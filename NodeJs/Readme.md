# Introdução ao Node.JS

### Verificar as versões do Node.JS
    
`node -v`

`npm -v`

### Iniciar o Projeto 

`npm init`

`npm install express`

### Rodar a Aplicação

`node index.js`

### Instalando EJS
    
`npm install ejs`

### Instalar Nodemon
    
`npm install nodemon` 
>Reinicia o servidor automaticamente
>>Criar um `script` no `package.json` 
>>> `"start": "npx nodemon index.js"`
